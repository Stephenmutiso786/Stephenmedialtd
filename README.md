# Tech Hub Portal\n\nA modern neon-themed web portal built with Next.js, Node.js (API), and MongoDB.\n\n## Features\n- Authentication (register/login)\n- User profiles\n- Dashboard\n- Content management (CRUD)\n- Analytics page\n- Messaging/chat\n- Random Joke Generator (external API)\n- Neon green tech theme and logo branding\n\n## Getting Started\n\n1. Clone the repo  \n   ```\n   git clone https://github.com/Stephenmutiso786/Stephenmedialtd.git\n   cd Stephenmedialtd\n   ```\n\n2. Install dependencies  \n   ```\n   npm install\n   ```\n\n3. Add your environment variables  \n   - Copy `.env.example` to `.env`\n   - Fill in your MongoDB URI and JWT secret\n\n4. Add your logo image  \n   - Place your logo file as `/public/logo.png`\n\n5. Start the development server  \n   ```\n   npm run dev\n   ```\n\n## Publish/Deploy\n\n- **Vercel** (Recommended for Next.js):\n  - Import your repo into Vercel and add your environment variables.\n- **Heroku** (if using backend APIs outside Next.js):\n  - Deploy with Heroku, set environment variables in dashboard.\n\n## More\n\n- Customize features/pages/components as needed!\n- For support, contact @Stephenmutiso786 on GitHub.
